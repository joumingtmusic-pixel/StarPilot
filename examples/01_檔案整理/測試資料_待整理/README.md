## 🎯 練習目標

使用這些檔案來練習：

### 練習 1：基本分類（對應任務說明的情境 1）
```
把檔案分成三類：
- 文件資料夾（PDF 和 Word 檔案）
- 圖片資料夾（JPG 和 PNG 檔案）
- 影片資料夾（MP4 檔案）
```

**對話範例**：
```
👤：幫我整理測試資料_待整理資料夾
    把 PDF 和 Word 放到「文件」資料夾
    把 JPG 和 PNG 放到「圖片」資料夾
    把 MP4 放到「影片」資料夾

🤖：了解，我會建立這個分類工具...
```

### 練習 2：按日期整理（對應任務說明的情境 2）
```
在每個類別下，再按照月份建立子資料夾
例如：
文件/
  ├── 2024-01/
  ├── 2024-03/
圖片/
### 練習 4：安全預覽（對應任務說明的情境 3）
```
先預覽會移動哪些檔案，確認後再執行
```

**對話範例**：
```
👤：等等，我想先看看會移動哪些檔案，不要真的移動

🤖：了解，我會先產生一個預覽清單...
    
    預計移動的檔案：
    📄 2024年度銷售報告.pdf → 文件/
    📄 會議記錄_0315.pdf → 文件/
    🖼️ 產品照片_001.jpg → 圖片/
    ...
```

### 練習 5：重新命名
```
統一檔名格式，加上日期前綴：
原本：會議記錄_0315.pdf
改成：2024-03-15_會議記錄.pdf

原本：team_photo_2024.jpg
改成：2024_團隊合照.jpg

## 💡 使用方式

1. **備份原始檔案**（第一次練習前）
   ```
   複製整個資料夾作為備份
   ```

2. **開始對話練習**
   ```
   開啟 Copilot Chat (Ctrl+I)
   說：「幫我整理 測試資料_待整理 資料夾」
   然後逐步描述你的需求
   ```

3. **驗證結果**
   ```
   檢查檔案是否正確分類
   確認沒有遺漏或錯誤
   ```

4. **重置練習**
   ```
   如果想重新練習，從備份還原檔案即可
   ```

## 🎓 學習重點

- **清楚描述需求**：越具體越好
- **逐步細化**：先大方向，再補充細節
- **確認理解**：讓 AI 重述你的需求
- **安全優先**：先預覽再執行
- **隨時調整**：發現不對立刻修正
