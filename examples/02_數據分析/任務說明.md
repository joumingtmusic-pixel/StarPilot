## 💡 提示

### 資料分析的對話技巧

**好的描述**：
```
👤：分析 data/sales.xlsx 檔案
    計算每個產品的平均銷售額
    並用長條圖顯示前 10 名
```

**不好的描述**：
```
👤：分析資料 ❌（什麼資料？分析什麼？）
```

### 常用分析類型

1. **統計摘要**：平均值、中位數、標準差
2. **排名分析**：Top N、Bottom N
3. **趨勢分析**：時間序列、成長率
4. **分布分析**：直方圖、圓餅圖
5. **關聯分析**：散佈圖、相關係數

### 圖表選擇建議

- **比較數值** → 長條圖 (Bar Chart)
- **顯示趨勢** → 折線圖 (Line Chart)
- **比例分配** → 圓餅圖 (Pie Chart)
- **分布情況** → 直方圖 (Histogram)
- **多維關係** → 散佈圖 (Scatter Plot)

## 🌟 小技巧

### 如何描述圖表需求

**明確指定圖表類型**：
```
👤：用長條圖顯示各產品銷售額
👤：用折線圖呈現每月趨勢
👤：用圓餅圖顯示地區分布
```

### 如何優化圖表

```
👤：圖表太小了，可以放大嗎？
👤：可以改成橫向長條圖嗎？
👤：顏色能改成藍色系嗎？
👤：圖例能放在右邊嗎？
```

### 如何處理大量資料

```
👤：資料太多了，可以只顯示前 20 筆嗎？
👤：可以按月份彙總嗎？
👤：能不能篩選出金額大於 10000 的記錄？
```

記住：**你不需要懂統計公式或程式語法，只要清楚描述你想要的分析結果！** 📊
