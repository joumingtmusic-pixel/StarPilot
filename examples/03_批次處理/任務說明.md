# ⚙️ 批次處理任務

## 🎯 任務目標

學習如何用自然語言讓 AI 幫你批次處理檔案：重新命名、格式轉換、內容修改。

---

## 💬 與 AI 的對話範例

### 情境 1：批次重新命名

```
👤：我有一堆照片檔案，檔名很亂
    想統一改成：旅遊_001.jpg、旅遊_002.jpg 這樣

🤖：好的！請問這些照片在哪個資料夾？

👤：在測試資料/待重新命名資料夾

🤖：了解，我會幫你：
    1. 找出所有 JPG 檔案
    2. 按照檔案建立時間排序
    3. 重新命名為 旅遊_001.jpg、旅遊_002.jpg...
    
    正在生成程式...
    [執行完成]
    
    ✅ 已重新命名 15 個檔案
    
    預覽：
    IMG_1234.jpg → 旅遊_001.jpg
    DSC_5678.jpg → 旅遊_002.jpg
    ...
```

## 🎓 學到的技巧

1. **批次思維**：一次處理大量檔案，而不是一個一個手動處理
2. **規則定義**：清楚說明命名規則或處理邏輯
3. **安全優先**：重要操作前先備份或預覽
4. **自動化**：把重複性工作變成可重複使用的工具

## 💡 提示

### 批次重新命名的技巧

**明確指定命名規則**：
```
✅ 好的描述：
「改成：會議記錄_2024-01_001.txt
 格式是：會議記錄_年月_流水號」

❌ 不好的描述：
「改成有規則的檔名」（什麼規則？）
```

### 格式轉換注意事項

```
👤：轉換時要注意什麼？

🤖：建議先確認：
1. 是否保留原檔案？
2. 轉換品質要求？（壓縮比例）
3. 輸出資料夾位置？
4. 檔名是否保持不變？
```

### 批次處理前的安全確認

```
👤：可以先預覽會改什麼嗎？

🤖：當然！我會先列出：
- 要處理的檔案清單
- 每個檔案的變更內容
- 預估處理時間

確認後再執行
```

## 🌟 小技巧

### 如何描述複雜的命名規則

**範例 1：包含多個元素**
```
👤：檔名格式：部門_日期_流水號.副檔名
    例如：業務部_2024-03-20_001.pdf
```

**範例 2：條件式命名**
```
👤：如果檔案大於 5MB，加上「大檔」前綴
    如果檔案小於 1MB，加上「小檔」前綴
```

### 批次處理的黃金法則

1. **先測試一個檔案**：確保邏輯正確
2. **一定要備份**：重要檔案處理前先備份
3. **分階段執行**：大量檔案分批處理
4. **記錄變更**：保存處理日誌方便追蹤

### 常見批次處理類型

- **重新命名**：統一命名格式
- **格式轉換**：圖片、文件、影片格式
- **內容修改**：批次查找替換
- **屬性修改**：日期、權限、標籤
- **檔案分類**：按規則移動或複製
