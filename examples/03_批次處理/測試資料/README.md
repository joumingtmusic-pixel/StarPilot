# 🧪 批次處理測試資料說明

這個資料夾包含用於練習批次處理的各種測試檔案。

---

## 📁 資料夾結構

### 1. 待重新命名/
- **用途**：練習批次重新命名檔案
- **檔案數**：10 張圖片
- **檔案類型**：JPG 圖片檔
- **特點**：檔名混亂，包含各種命名規則
  - IMG_xxxx.jpg（相機命名）
  - DSC_xxxx.jpg（相機命名）
  - PHOTO_xxx.jpg（手機命名）
  - PIC_日期.jpg（日期命名）
  - snapshot_xx.jpg（截圖命名）

**練習目標**：
```
將混亂的檔名統一改成：
產品照片_001.jpg
產品照片_002.jpg
產品照片_003.jpg
...
```

---

### 2. 圖片資料/
- **用途**：練習圖片格式轉換和批次壓縮
- **檔案數**：5 個 PNG 檔案
- **檔案類型**：PNG 圖片
- **特點**：檔案較大，適合練習壓縮和格式轉換

**練習目標**：
```
1. 批次轉換：PNG → JPG
2. 壓縮圖片：減少檔案大小
3. 調整尺寸：統一圖片大小
```

---

## 🎯 練習建議

### 🥇 初級練習：批次重新命名

用 Copilot Chat 對話：
```
「把 待重新命名/ 資料夾裡的所有圖片
 重新命名成：產品照片_001.jpg, 產品照片_002.jpg...
 按照檔案建立時間排序」
```

---

### 🥈 中級練習：格式轉換

```
「將 圖片資料/ 裡的所有 PNG 檔案
 轉換成 JPG 格式
 並壓縮到 80% 品質
 儲存到新資料夾 converted/」
```

---

### 🥉 高級練習：綜合批次處理

```
「建立一個批次處理工具：
1. 讀取資料夾裡所有圖片
2. 重新命名（加上日期前綴）
3. 調整大小到 1920x1080
4. 轉換成 JPG 格式
5. 壓縮到 85% 品質
6. 儲存到 output/ 資料夾」
```

---

## 💡 安全提示

在執行批次處理前，記得：

1. **先備份**：複製一份原始檔案
2. **先測試**：先在一個檔案上測試
3. **檢查結果**：處理完後檢查是否正確
4. **保留記錄**：記錄處理前後的檔名對照

## 🚀 進階應用

### 應用 1：批次加浮水印
```
「在所有圖片右下角加上浮水印
 文字：© 2024 公司名稱」
```

### 應用 2：按規則分類
```
「根據檔案建立日期
 自動分類到不同資料夾
 2024-01 → January/
 2024-02 → February/」
```

### 應用 3：定時批次處理
```
「建立自動化腳本
 每天晚上 11 點
 自動處理 input/ 資料夾的新圖片」
```
