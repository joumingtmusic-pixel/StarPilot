# 📚 測試資料說明

這個資料夾包含用於練習自動化報告的測試數據：

## 📊 sales_data.csv
- **用途**：銷售數據分析
- **記錄數**：11 筆
- **欄位**：
  - 日期：2024-03-01 ~ 2024-03-10
  - 產品：筆記型電腦、滑鼠、鍵盤、螢幕、耳機
  - 數量：銷售數量
  - 金額：銷售金額（新台幣）
  - 地區：台北、台中、高雄、台南
  - 業務：負責業務員姓名

## 🎯 練習建議

### 初級練習
1. 讀取資料並計算總銷售額
2. 找出銷售金額最高的產品
3. 計算各地區的銷售總額

### 中級練習
1. 生成 Excel 報告，包含：
   - 產品銷售排名
   - 地區銷售統計
   - 業務績效表
2. 加入圖表：
   - 產品銷售長條圖
   - 地區分布圓餅圖
   - 每日銷售趨勢折線圖

### 高級練習
1. 生成完整的 PowerPoint 簡報
2. 輸出專業的 PDF 報告
3. 建立自動化週報系統
